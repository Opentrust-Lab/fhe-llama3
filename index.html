Cipher LLaMA3 model

  • Nx stands for repeat N layers

  • stands for Residual connection

  • The Tai Chi symbol represents rotation position encoding, and position encoding information is added to q and k before the self attention operation of each layer

  • Self attention mechanism uses group query attention mechanism and utilizes KV cache caching

Model Architecture

  1. LLaMA 3 still uses the decoder only transformer architecture
  2. LLaMA 3 uses 128K token vocabulary (LLaMA 2 is 32K), which improves encoding efficiency and significantly enhances model performance.
  3. Group Query Attention (GQA) improved inference efficiency
  4. Train on a token sequence with a length of 8192 (previously 4K) and use masks to ensure that self attention does not cross document boundaries.
